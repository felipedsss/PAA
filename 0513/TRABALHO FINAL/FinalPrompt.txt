Descrição da Biblioteca QDB - Queue DDLL BST

A biblioteca QDB é uma implementação que combina uma Queue Dynamic (QD), uma Dynamic Doubly Linked List (DDLL) e uma Binary Search Tree (BST) para fornecer uma estrutura de dados eficiente para armazenar e gerenciar índices e dados associados. A QDB é projetada para ser flexível, eficiente em termos de espaço e tempo, e fácil de usar, permitindo a inserção, busca, remoção e impressão de elementos de forma rápida e eficiente. A Figura 1 ilustra um exemplo de dados armazenados na QDB.

Componentes da Biblioteca:

Queue Dynamic (QD): A QD é uma estrutura de fila dinâmica que armazena índices gerados aleatoriamente. Esta estrutura é implementada usando a Dynamic Doubly Linked List (DDLL) para permitir inserção e remoção eficientes tanto no início quanto no final da fila.

Interfaces Públicas da DDLL (DISPONÍVEL PARA USO):

int cDDLL(ppDDLL pp, int sizedata): Cria uma DDLL, alocando memória e inicializando as estruturas necessárias.
int dDDLL(ppDDLL pp): Destroi a DDLL, liberando toda a memória alocada para suas estruturas internas.
int cleanDDLL(pDDLL p): Limpa a DDLL, removendo todos os elementos, mas mantendo a estrutura intacta.
int iBegin(pDDLL p, void *element): Insere um novo elemento no início da DDLL.
int iEnd(pDDLL p, void *element): Insere um novo elemento no final da DDLL.
int iPosition(pDDLL p, int N, void *element): Insere um novo elemento em uma posição específica da DDLL.
int rBegin(pDDLL p, void *element): Remove o elemento no início da DDLL.
int rEnd(pDDLL p, void *element): Remove o elemento no final da DDLL.
int rPosition(pDDLL p, int N, void *element): Remove o elemento em uma posição específica da DDLL.
int sBegin(pDDLL p, void *element): Busca o elemento no início da DDLL.
int sEnd(pDDLL p, void *element): Busca o elemento no final da DDLL.
int sPosition(pDDLL p, int N, void *element): Busca o elemento em uma posição específica da DDLL.
int empty(pDDLL p): Verifica se a DDLL está vazia.
int countElements(pDDLL p): Conta o número de elementos na DDLL.
Binary Search Tree (BST): A BST é utilizada para armazenar os índices como chaves de busca associadas aos dados fornecidos pelo usuário. Isso permite uma busca eficiente dos dados associados aos índices, garantindo um desempenho rápido e eficiente.

Interfaces Públicas da BST (DISPONÍVEL PARA USO):

int cBST(ppBST pp, int sizedata): Cria uma BST, alocando memória e inicializando as estruturas necessárias.
int dBST(ppBST pp): Destroi a BST, liberando toda a memória alocada para suas estruturas internas.
int cleanBST(pBST p): Limpa a BST, removendo todos os elementos, mas mantendo a estrutura intacta.
int iBST(pBST p, void *new, int (* cmp)(void *p1, void *p2)): Insere um novo elemento na BST.
int rBST(pBST p, void *item, int (* cmp)(void *p1, void *p2)): Remove um elemento da BST.
int sBST(pBST p, void *item, int (* cmp)(void *p1, void *p2)): Busca um elemento na BST.
int emptyBST(pBST p): Verifica se a BST está vazia.
int orderPath(pBST pa, void (* process)(void *p)): Percorre a BST em ordem e aplica uma função de processo a cada nó.
int preorderPath(pBST pa, void (* process)(void *p)): Percorre a BST em pré-ordem e aplica uma função de processo a cada nó.
int postorderPath(pBST pa, void (* process)(void *p)): Percorre a BST em pós-ordem e aplica uma função de processo a cada nó.
Funções Públicas da QDB:

int cQDB(ppQDB pp, int sizedata): Esta função é responsável por criar a QDB, alocando memória e inicializando todas as estruturas necessárias. O parâmetro pp é um ponteiro para um ponteiro para a QDB, que será inicializado por esta função. sizedata especifica o tamanho dos dados que serão armazenados na QDB.

int dQDB(ppQDB pp): Esta função destroi a QDB, liberando toda a memória alocada para suas estruturas internas. O parâmetro pp é um ponteiro para um ponteiro para a QDB que será destruída.

int iQDB(pQDB p, void *new): Esta função insere um novo elemento na QDB. O parâmetro p é um ponteiro para a QDB, e new é um ponteiro para os dados que serão inseridos na QDB.

int sQDB(pQDB p, void *new): Esta função busca um elemento na QDB com base no índice fornecido. O parâmetro p é um ponteiro para a QDB, e new é um ponteiro para os dados associados ao índice a ser buscado.

int rQDB(pQDB p, void *new): Esta função remove um elemento da QDB com base no índice fornecido. O parâmetro p é um ponteiro para a QDB, e new é um ponteiro para os dados associados ao índice a ser removido.

int prQDB(pQDB p, void (*printall)(void *p)): Esta função imprime todos os elementos armazenados na QDB. O parâmetro p é um ponteiro para a QDB, e printall é um ponteiro para uma função de impressão que será aplicada a cada elemento da QDB para imprimir seus dados.


Dicas:

Estrutura Descritora da QDB:

A QDB é uma estrutura descritora que encapsula uma Queue Dynamic (QD), baseada na Dynamic Doubly Linked List (DDLL), e uma Binary Search Tree (BST). 
Armazenamento de Dados:

Os dados recebidos do usuário serão encapsulados em um pacote que inclui a chave (índice) e os dados propriamente ditos. Esses pacotes serão armazenados na BST.
Além disso, o índice também será armazenado na QD, seguindo as regras de uma fila clássica, nas quais a inserção ocorre no final e a remoção ou busca no início.
Acesso aos Dados na BST:

Para acessar os dados armazenados na BST, siga as regras da QD. Isso significa que as funções sQDB e rQDB devem remover ou buscar o dado no início da QD para, em seguida, encontrar a chave correspondente na BST e, por fim, retornar o dado original do usuário.
Impressão por Ordem de Chave na BST:

Utilize a função orderPath da BST para imprimir os dados em ordem de chave. Essa função requer uma função de impressão como parâmetro, já que a estrutura BST desconhece o formato dos dados armazenados.
A função de impressão deve ser definida na aplicação, pois somente ela conhece o formato dos dados. No entanto, lembre-se de que os dados armazenados na BST não consistem puramente nos dados recebidos pelo usuário, pois a QDB acrescenta o índice (chave) ao pacote armazenado.
Declaração de Ponteiro para Função:

Considere declarar um ponteiro para a função recebida do usuário no QDB_priv. Isso permitirá que a biblioteca QDB desempacote o dado recebido da BST, removendo a chave e, posteriormente, chame a função de impressão do próprio usuário com o dado original.
Bibliotecas disponíveis (DDLL e BST):
Observe o arquivo QDB_priv.h que já inclui as interfaces da funções das bibliotecas DDLL e BST além dos ponteiros para os descritos das referidas bibliotecas.

#include <stdlib.h>

// Comparação de nós da QDB
int compareQDBNode(void *p1, void *p2) {
    pacote *a = (pacote *)p1;
    pacote *b = (pacote *)p2;
    if (a->index < b->index) return -1;
    if (a->index > b->index) return 1;
    return 0;
}

// Criação da QDB
int cQDB(ppQDB pp, int sizedata) {
    (*pp) = (pQDB)malloc(sizeof(QDBST));
    if (*pp == NULL)
        return FAIL;
    
    (*pp)->sizedata = sizedata;

    if (cDDLL(&(*pp)->qd, sizeof(int)) != SUCCESS) {
        free(*pp);
        return FAIL;
    }
    (*pp)->bst = NULL;
    if (cBST(&(*pp)->bst, sizeof(pacote)) != SUCCESS) {
        dDDLL(&(*pp)->qd);
        free(*pp);
        return FAIL;
    }

    return SUCCESS;
}

// Destruição da QDB
int dQDB(ppQDB pp) {
    if (pp == NULL || *pp == NULL)
        return FAIL;

    //limpar todos os dados alocados no pacote na função iBST
    int intret;
    while (rQDB((*pp), &intret) == SUCCESS) {
        continue;
    }
    //remover a árvore e a fila DDLL em seguida
    dBST(&(*pp)->bst);
    dDDLL(&(*pp)->qd);
    free(*pp);
    *pp = NULL;
    return SUCCESS;
}

// Inserção na QDB
int iQDB(pQDB p, void *new) {
    if (p == NULL || new == NULL)
        return FAIL;
    int index = rand();
    if (iEnd(p->qd, &index) != SUCCESS) {
        return FAIL;
    }
    //Criar pacote com memória alocada para receber os dados
    pacote newPacote;
    newPacote.index = index;
    newPacote.data = (void *)malloc(p->sizedata);
    memcpy(newPacote.data, new, p->sizedata);
       

    if (iBST(p->bst, &newPacote, compareQDBNode) != SUCCESS) {
        free(newPacote.data);
        rEnd(p->qd, NULL);  // Removing the index from the queue
        return FAIL;
    }

    return SUCCESS;
}

// Busca na QDB
int sQDB(pQDB p, void *new) {
    if (p == NULL || new == NULL) {
        fprintf(stderr, "sQDB: Invalid input pointers.\n");
        return FAIL;
    }
    
    //Encontrar o índice no elemento inicial da fila
    int index;

    if (sBegin(p->qd, &index) != SUCCESS) {
        return FAIL;
    }
    //Criar um pacote com o índice encontrado, ele receberá os dados do respectivo nó na BST
    pacote buscaPacote;
    buscaPacote.index = index;

    //Encontrar o pacote no BST que possui o índice encontrado
    if (sBST(p->bst, &buscaPacote, compareQDBNode) != SUCCESS) {
        return FAIL;
    }
    
    memcpy(new, buscaPacote.data, p->sizedata);
    return SUCCESS;
}

// Remoção da QDB
int rQDB(pQDB p, void *new) {
    if (p == NULL || new == NULL) {
        fprintf(stderr, "rQDB: p or new is NULL.\n");
        return FAIL;
    }

    int index;
    if (rBegin(p->qd, &index) != SUCCESS) {
        return FAIL;
    }
    
    pacote buscaPacote;
    buscaPacote.index = index;

    if (rBST(p->bst, &buscaPacote, compareQDBNode) != SUCCESS) {
        fprintf(stderr, "rQDB: rBST failed.\n");
        return FAIL;
    }
    
    memcpy(new, buscaPacote.data, p->sizedata);
    free(buscaPacote.data);
    return SUCCESS;
}

// Impressão da QDB
int prQDB(pQDB p, void (*printall)(void *p)) {
    printf("[Index]\t(User data)\n");

    if (p == NULL || printall == NULL)
        return FAIL;

    return orderPath(p->bst, printall);
}
